# Adobe Workfront MCP 서버 개요

[!DNL Adobe Workfront] MCP(Model Context Protocol) 서버를 사용하면 지원되는 AI 도우미에서 자연어 대화를 통해 Workfront 데이터로 작업할 수 있습니다. Workfront을 클릭하여 프로젝트를 찾거나 작업을 업데이트하거나 승인을 확인하는 대신 일반 영어로 AI 도우미에 요청하면 AI 도우미가 작업을 대신 수행합니다.

Workfront MCP 서버를 사용하기 위해 Workfront API 또는 MCP에 대해 알 필요가 없습니다. Workfront과 한 번 대화한 다음 질문하도록 AI 비서를 설정하십시오.

## Workfront MCP 서버 설정 및 사용

Workfront MCP 서버를 사용하려면 먼저 Workfront 관리자가 Workfront에서 MCP 서버 액세스를 허용해야 하며 AI 비서를 Workfront에 연결해야 합니다. 자세한 내용은 [Adobe Workfront MCP 서버 구성](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md)을 참조하십시오.

설정 후 AI 도우미에게 Workfront 항목을 검색, 생성, 업데이트 및 관리하도록 요청할 수 있습니다. 사용 가능한 작업, 요청 및 데이터 처리에 대한 자세한 내용은 [Adobe Workfront MCP 서버 사용](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md)을 참조하세요.
