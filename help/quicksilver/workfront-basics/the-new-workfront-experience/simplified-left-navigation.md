# 에서 왼쪽 탐색 [!DNL Adobe Workfront]

WF의 대부분의 영역과 개체는 화면 왼쪽에 있는 간단한 탐색 패널을 사용합니다. 왼쪽 패널 탐색의 장점은 다음과 같습니다.

* 화면 공간을 더 잘 관리할 수 있습니다.
* 사용자 [!DNL Workfront] 관리자는 레이아웃 템플릿을 사용하여 객체의 세부 사항을 제외하고 왼쪽 패널에서 모든 섹션을 숨길 수 있습니다.

   레이아웃 템플릿 사용에 대한 자세한 내용은 문서를 참조하십시오 [레이아웃 템플릿 만들기 및 관리](../../administration-and-setup/customize-workfront/use-layout-templates/create-and-manage-layout-templates.md).

* 원하는 순서로 끌어서 놓아 추가 화면을 열지 않고 왼쪽 탐색 영역에서 섹션의 순서를 쉽게 변경할 수 있습니다.

   자세한 내용은 이 문서에서 다음 섹션을 참조하십시오. [왼쪽 탐색 패널 사용](#use-the-left-navigation-panel).

* 대시보드를 추가하여 왼쪽 패널을 사용자 지정할 수 있습니다.

## 왼쪽 탐색 패널의 기본 섹션

Adobe Workfront의 다양한 개체 또는 영역에 대해 왼쪽 패널에서 개체나 영역에 대한 자세한 정보에 액세스할 수 있습니다.

탐색한 객체 또는 영역에 따라 왼쪽 패널에 나타나는 섹션이 달라집니다.

왼쪽 패널은 다음 객체에 사용할 수 있습니다.

* 프로젝트
* 작업
* 문제
* 포트폴리오
* 프로그램
* 템플릿
* 템플릿 작업
* 반복
* 사용자
* 팀
* 그룹
* 목표

>[!IMPORTANT]
>
>왼쪽 패널에 기본적으로 다음과 같은 하나의 섹션만 표시되는 경우 **[!UICONTROL 모든 프로젝트]** 에서 **[!UICONTROL 프로젝트]** 영역 - [!DNL Workfront] 관리자는 레이아웃 템플릿을 작성할 때 이 영역에 하나 이상의 사용자 지정 섹션을 추가하고 왼쪽 패널이 해당 영역에 표시되기 전에 해당 템플릿에 사용자를 할당해야 합니다.\
>자세한 내용은 [!DNL Workfront] 관리자는 레이아웃 템플릿의 왼쪽 패널을 사용자 지정합니다. [레이아웃 템플릿을 사용하여 왼쪽 패널 사용자 정의](../../administration-and-setup/customize-workfront/use-layout-templates/customize-left-panel.md).

왼쪽 패널은 다음 영역에 사용할 수 있습니다.

* [의 기본 섹션 [!UICONTROL 대시보드] 영역](#default-sections-in-the-dashboards-area)
* [의 기본 섹션 [!UICONTROL 요청] 영역](#default-sections-in-the-requests-area)
* [의 기본 섹션 [!UICONTROL 리소스] 영역](#default-sections-in-the-resourcing-area)
* [의 기본 섹션 [!UICONTROL 프로젝트] 영역](#default-sections-in-the-projects-area)
* [의 기본 섹션 [!UICONTROL 작업표] 영역](#default-sections-in-the-timesheets-area)
* [의 기본 섹션 [!DNL Goals] 영역](#default-sections-in-the-goals-area)

## 왼쪽 탐색 패널 사용

왼쪽 패널에서 정보를 찾고 편집하는 방법은 이러한 객체에 대해 유사합니다. 왼쪽 패널에서 사용할 수 있는 옵션은 액세스하는 객체에 따라 다릅니다.

특정 영역에 사용할 수 있는 섹션을 알아보려면 [[!UICONTROL 기본 섹션] 왼쪽 탐색 패널에서](#default-sections-in-the-left-navigation-panel).

1. 을(를) 클릭합니다. **[!UICONTROL 기본 메뉴]** 아이콘 ![](assets/main-menu-icon-16x12.png)를 클릭한 다음 개체의 이름을 클릭하여 액세스합니다.

   객체 페이지가 표시됩니다.

1. (조건부) 객체 목록이 포함된 영역을 선택한 경우(예: **[!UICONTROL 프로젝트]** 또는 **[!UICONTROL Portfolio]**- 왼쪽 패널 탐색에 액세스하려면 다음을 수행해야 합니다.

   1. 목록에서 개체를 클릭합니다.
   1. (선택 사항) 왼쪽 패널 탐색에서 다음과 같은 다른 개체를 선택할 수 있습니다 **[!UICONTROL 작업]** 또는 **[!UICONTROL 문제]** 프로젝트 내 - 왼쪽 패널에서 객체 목록에 액세스한 다음 특정 객체의 이름을 클릭합니다.\

      객체 페이지가 표시됩니다.

1. 개체 페이지에서 왼쪽 패널의 섹션을 클릭하여 해당 섹션의 정보를 보거나 편집합니다.

   정보를 편집하려면 먼저 개체를 편집할 수 있는 권한이 있어야 합니다.

   ![왼쪽 패널 섹션](assets/NWE-left-navigation-panel.png)

1. (선택 사항) **[!UICONTROL 자세히 표시]** 왼쪽 패널 하단에 더 많은 섹션을 표시하려면 을 클릭하십시오.

   >[!NOTE]
   >
   >이 옵션은 기본적으로 왼쪽 패널에 모든 섹션이 표시되는 경우에는 사용할 수 없습니다.

1. (선택 사항) 대시보드 빠른 링크를 추가하려면 **[!UICONTROL 대시보드 추가]** 왼쪽 패널 아래에서 다음을 수행합니다.

   1. 대시보드의 이름을 **[!UICONTROL 빠른 링크 이름]** 필드.
   1. 에 기존 대시보드의 이름을 입력합니다. **[!UICONTROL 대시보드 선택]** 필드를 클릭한 다음 목록에 표시할 때 대시보드를 클릭합니다.

      >[!TIP]
      >
