## 보기 [!UICONTROL 요약] 작업 또는 문제 목록의 패널

1. 작업이나 문제로 이동하여 목록에서 항목을 선택합니다.
1. 을(를) 클릭합니다. **[!UICONTROL 요약]** 아이콘 ![](assets/qs-summary-in-new-toolbar-small.png)

   또는

   을(를) 클릭합니다. **[!UICONTROL 요약 열기]** 아이콘 ![](assets/open-summary-with-text-nwe.png) 에서 [!UICONTROL 제출됨] 섹션 [!UICONTROL 요청] 영역.

   요약 을 연 후에는 을 클릭하거나 다른 작업이나 문제를 선택할 때 열려 있고 수동으로 닫을 때까지 열려 있습니다.

   >[!TIP]
   >
   >작업 또는 문제를 한 번에 하나만 선택하여 [!UICONTROL 요약] 패널.

   ![](assets/summary-overview--open-task-from-summary-icon-350x112.png)

1. (선택 사항) [!UICONTROL 요약] 패널에서 다음 중 하나를 수행합니다.

   * 작업 또는 문제 목록에서 **[!UICONTROL 요약 열기]** 아이콘 ![](assets/summary-panel-icon.png)

      또는

      을(를) 클릭합니다. **X** 아이콘( [!UICONTROL 요약] 패널.

   * 에서 [!UICONTROL 제출됨] 섹션 [!UICONTROL 요청] 영역에서 **[!UICONTROL 요약 닫기]** 아이콘 ![](assets/close-summary-with-text-nwe.png)

      또는

      을(를) 클릭합니다. **X** 요약 패널의 오른쪽 위 모서리에 있는 아이콘.

## [!UICONTROL 완료율]

맨 위에 있는 진행률 표시줄 사용 [!UICONTROL 요약] 선택한 작업 또는 문제에 대한 완료 비율을 업데이트하려면 숫자를 입력하거나 막대를 올바른 백분율로 드래그합니다.

![](assets/summary-overview-percent-complete-350x395.png)

## [!UICONTROL 업데이트]

를 사용하십시오 [!UICONTROL 업데이트] 섹션 [!UICONTROL 요약] 최근 업데이트를 보고 선택한 작업 또는 문제에 대한 업데이트를 수행하려면 다음을 수행하십시오. 클릭 **[!UICONTROL 모두 보기]** 로 바로 이동 [!UICONTROL 업데이트] 탭에서 작업을 수행합니다.

![](assets/summary-updates-with-block-quote-350x290.png)

## [!UICONTROL 문서]

를 사용하십시오 [!UICONTROL 문서] 섹션 [!UICONTROL 요약] 선택한 작업 또는 문제에 첨부된 문서를 보려면 축소판을 클릭하여 문서 미리 보기를 엽니다. 로 바로 이동 [!UICONTROL 문서] 작업이나 문제의 탭에서 **[!UICONTROL 문서]** 제목.

![](assets/summary-overview-documents-350x84.png)

## [!UICONTROL 세부 정보]

를 사용하십시오 [!UICONTROL 세부 사항] 섹션 [!UICONTROL 요약] 상위 레벨 작업 품목 상세내역을 조회하려면 지정을 하거나 시작 일자를 추가합니다. 클릭 **[!UICONTROL 모두 보기]** 로 바로 이동 [!UICONTROL 세부 사항] 탭에서 작업을 하거나 문제를 확인합니다.

>[!NOTE]
>
>이 섹션에 표시되는 필드는 홈 의 오른쪽 패널에 표시되는 필드와 같습니다. 이러한 필드를 사용자 지정할 수 있습니다 [사용자 지정 [!UICONTROL 홈] 및 [!UICONTROL 요약] 레이아웃 템플릿 사용](../../administration-and-setup/customize-workfront/use-layout-templates/customize-home-summary-layout-template.md).

![](assets/summary-overview-details-new-350x278.png)

## [!UICONTROL 하위 작업]

이 섹션은 작업에만 사용할 수 있습니다. 를 사용하십시오 [!UICONTROL 하위 작업] 섹션 [!UICONTROL 요약] 를 보려면 [!UICONTROL 새로 만들기], [!UICONTROL 진행 중], 및 [!UICONTROL 닫힘] 선택한 작업의 하위 작업입니다. 을(를) 클릭합니다. **[!UICONTROL 상태]** 상태 간에 전환할 드롭다운 메뉴 로 바로 이동 [!UICONTROL 하위 작업] 작업의 탭에서 **[!UICONTROL 하위 작업]**&#x200B;제목&#x200B;.

작업에 하위 작업을 추가하지 않은 경우 **[!UICONTROL 여기에 추가]** 로 바로 이동 [!UICONTROL 하위 작업] 탭에서 작업을 수행합니다.

![](assets/summary-overview-subtasks-350x140.png)

## [!UICONTROL 시간]

를 사용하십시오 [!UICONTROL 시간] 섹션 [!UICONTROL 요약] 선택한 작업 또는 문제에 대해 시간을 기록하려면 다음을 수행하십시오. 클릭 **[!UICONTROL 로그 시간]** 시간을 입력합니다. 작업이나 문제의 시간 탭으로 직접 이동하려면 **[!UICONTROL 시간]** 제목.

의 시간 카운트 [!UICONTROL 요약] 기록하는 시간을 표시합니다. 다른 사용자의 경우 [!UICONTROL 요약] 작업에 로그온한 시간에 따라

계획이 없는 경우 [!UICONTROL 시간] 작업 또는 문제에 로그된 시간이 있으면 시간 표시줄이 빨간색으로 표시됩니다.

![](assets/summary-overview-hours-350x96.png)
