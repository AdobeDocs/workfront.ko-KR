# 시작하기 [!UICONTROL 홈]

에서는 두 가지 방법으로 작업을 볼 수 있습니다 [!UICONTROL 홈] 영역(둘 중 하나) [!UICONTROL 작업 목록] 또는 달력

## 작업 목록

다음 [!UICONTROL 작업 목록] 할당된 작업을 한 위치에서 볼 수 있습니다.

![[!UICONTROL 작업 목록] 패널](assets/worklist-and-right-panel-home.png)

A. 왼쪽 패널을 사용하여 자신에게 할당된 작업을 필터링, 정렬 및 선택합니다.

B. 왼쪽 패널에서 작업 항목을 선택한 후 오른쪽 패널을 사용하여 사용자 지정 필드와 상호 작용하고 업데이트하며 로그 시간을 기록합니다.

사용 방법에 대한 자세한 정보 [!UICONTROL 홈 작업 목록], 다음 문서를 참조하십시오.

* [에 항목 표시 [!UICONTROL 작업 목록] 에서 [!UICONTROL 홈] 영역](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)
* [에서 작업 항목 만들기 [!UICONTROL 홈] 영역](../../../workfront-basics/using-home/using-the-home-area/create-work-items-in-home.md)

## 캘린더

달력을 사용하면 작업 항목이 기한인 시기를 표시하고 이를 완료하는 데 걸리는 시간을 차단할 수 있도록 하여 작업을 시각화할 수 있습니다.

![](assets/calendar-home-nwe-350x114.png)

A. [!UICONTROL 달력] 보기에 따라 완료해야 하는 작업의 시각적 맵을 만듭니다. 을 클릭하여 작업 항목을 [!UICONTROL 작업 목록] 을 눌러 작업 시간을 차단합니다.

B. 캘린더의 동기화 옵션을 사용하여 [!DNL Outlook] 달력. 모임을 관리하고 모든 작업을 한 곳에서 관리합니다.

C. [!UICONTROL 세부 사항] 작업 항목에 대한 자세한 정보를 볼 수 있는 새 오른쪽 패널을 여는 단추.

D. 사용자에게 지정된 작업 기한이 언제인지 추적하려면 기한 표시줄을 사용합니다.

홈 달력 사용 방법에 대한 자세한 내용은 [를 사용하십시오 [!UICONTROL 홈 달력] 보기](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md).

## 작업 찾기

다음 [!UICONTROL 홈] 영역은 사용자에게 할당된 작업, 문제 및 승인을 찾기 위한 원스톱 샵입니다. 필터 및 정렬 옵션은 [!UICONTROL 홈] 영역은 원하는 방식으로 작업할 수 있도록 함께 작동합니다. 를 사용할 수 있습니다 [!UICONTROL 정렬 기준] 항목을 함께 그룹화한 다음 필터를 사용하여 완료해야 하는 작업에 집중할 수 있는 옵션입니다.

>[!NOTE]
>
>필터 및 정렬 옵션은 브라우저에 저장됩니다. 동일한 컴퓨터에서 동일한 브라우저를 일관되게 사용(사이트 데이터를 지우지 않음)하면 필터 및 정렬이 변경되지 않지만 브라우저나 컴퓨터를 전환하면 필터 및 정렬이 달라집니다.

### 유사한 작업 항목을 &#39;[!UICONTROL 그룹화 기준]` 옵션

다음 [!UICONTROL 정렬 기준] 옵션을 사용하면 작업 목록에서 같은 항목을 그룹화할 수 있습니다. 다음을 기준으로 정렬할 수 있습니다.

* [!UICONTROL 계획된 완료]
* [!UICONTROL 계획된 시작]
* [!UICONTROL 커밋 일자]
* [!UICONTROL 프로젝트]
* [!UICONTROL 내 우선 순위]

그룹화 사용 방법에 대한 자세한 내용은 다음을 참조하십시오 [에 항목 표시 [!UICONTROL 작업 목록] 에서 [!UICONTROL 홈] 영역](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/group-by-drop-down-expanded-in-home-with-planned-start-date-nwe-350x273.png)

### 필터를 사용하여 포커스 좁히기

다음 [!UICONTROL 작업 목록] 필터를 사용하면 특정 작업 항목으로 포커스 범위를 좁힐 수 있습니다.

다음은 홈에서 보는 항목을 필터링할 수 있는 방법의 예입니다.

* 항목 유형 및 상태별로 필터링합니다.

   예를 들어, 모든 작업을 표시하려면 [!UICONTROL 작업] 필터. 작업을 시작할 준비가 된 작업만 표시하고 이 작업보다 구체화하려면 [!UICONTROL 시작 준비 완료] 아래의 필터 [!UICONTROL 작업] 필터.

* 항목 유형별로 필터링합니다.

   예를 들어 [!UICONTROL 문제] 모든 상태의 모든 문제를[!UICONTROL 작업] 또는 [!UICONTROL 요청]) 또는 [!UICONTROL 승인] 모든 작업 항목을 보려면 액세스 요청, 작업표, 문서 및 증명 승인을 확인하십시오.

* 상태로만 필터링합니다.

   예를 들어 [!UICONTROL 완료됨] 완료된 작업 및 문제를 표시하도록 필터링합니다. 여기에는 개인 작업이 포함되지만 승인을 포함하지 않습니다.

필터 사용 방법에 대한 자세한 내용은 [에 항목 표시 [!UICONTROL 작업 목록] 에서 [!UICONTROL 홈] 영역](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/displaying-work-items-filters-nwe-350x401.png)

### 팀에 할당된 작업에 액세스

다음 [!UICONTROL 홈] 영역에는 팀의 요청 페이지에 직접 연결되는 팀 요청을 위한 영구 그룹이 있습니다. 이 그룹을 사용하여 현재 있는 팀에 대한 요청을 보고 액세스할 수 있습니다.

>[!NOTE]
>
>다음 [!UICONTROL 필터] 및 [!UICONTROL 정렬 기준] 선택 사항은 영향을 주지 않습니다 [!UICONTROL 팀 요청] 그룹화. 이 그룹화는 팀에 할당된 작업만 있으면 표시됩니다.

팀 요청에 액세스하는 방법에 대한 자세한 내용은 문서를 참조하십시오 [[!UICONTROL 관리] 의 작업 및 팀 요청 [!UICONTROL 홈] 영역](../../../workfront-basics/using-home/using-the-home-area/manage-work-and-team-requests-home.md).

![](assets/team-requests-expanded-home-group-by-drop-down-nwe-350x314.png)

### 제출한 작업 모니터링

승인을 위해 직접 제출한 작업을 추적합니다. [!UICONTROL 작업 목록]. 승인자에게 승인이 필요한 작업에 대해 상기시킬 수 있습니다. 필요한 경우 승인을 회수할 수도 있습니다.

>[!NOTE]
>
>다음 [!UICONTROL 필터] 및 [!UICONTROL 정렬 기준] 선택 사항은 영향을 주지 않습니다 [!UICONTROL 승인 제출] 그룹화. 승인을 기다리는 작업이 있는 한 이 그룹이 표시됩니다.



![](assets/approvals-expanded-home-group-by-drop-down-nwe-350x415.png)

## 중요한 사항에 우선 순위 지정

다음 [!UICONTROL 작업 목록] 에서는 사용자에게 중요한 작업을 표면화하고 우선 순위를 지정할 수 있습니다. [!UICONTROL 내 우선 순위 정렬 기준] 선택 사항입니다. 에는 최대 20개의 항목을 추가할 수 있습니다 [!UICONTROL 내 우선 순위] 목록. 를 사용하십시오 [!UICONTROL 자세히] 메뉴를 사용하여 우선 순위 목록에 작업 항목을 추가할 수 있습니다.

>[!NOTE]
>
>에 추가된 모든 작업 항목 [!UICONTROL 내 우선 순위] 이것은 오직 당신에게만 표시되며, 아무도 당신에게 일을 우선시할 수 없습니다.

사용 방법에 대한 자세한 정보 [!UICONTROL 내 우선 순위]를 참조하십시오. [작업 우선 순위 지정 [!UICONTROL 홈] 영역](../../../workfront-basics/using-home/using-the-home-area/prioritize-work-in-home.md).

![](assets/prioritizing-in-home-group-by-drop-down-nwe-350x412.png)
