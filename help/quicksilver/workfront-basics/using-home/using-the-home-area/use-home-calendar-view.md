## 작업 항목을 [!UICONTROL 홈 달력]

를 사용할 수 있습니다 [!UICONTROL 홈 달력] 주의 작업 항목을 예약하려면 다음을 수행하십시오.

>[!NOTE]
>
>작업 항목을 [!UICONTROL 홈 달력] 작업 항목과 연관된 기존 날짜는 변경되지 않습니다.

을(를) 추가하려면 [!DNL Workfront] 달력에 작업 항목:

1. 로 이동합니다. [!UICONTROL 홈 달력] 에 설명된 대로 보기 [보기 [!UICONTROL 홈 달력]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).

   사용자 또는 사용자 그룹, 팀 또는 작업 역할에 할당된 모든 작업 항목이 왼쪽 패널에 표시됩니다. 목록을 정렬하여 특정 항목만 표시하도록 필터링할 수 있습니다.

   자세한 내용은 [작업 항목 필터링 및 정렬](#filter-and-sort-work-items) 이 문서의 뒷부분에 있습니다.

1. 왼쪽 패널에서 작업 항목을 작업할 날짜와 시간으로 드래그합니다.

   계획 완료 일자 이후에 또는 계획된 시간보다 더 많은 시간 동안 품목에 대해 작업하도록 예약하면 작업 항목이 빨간색으로 표시되고 경고가 표시됩니다.

   작업 품목 납기 일자 및 작업 품목 만기 일자가 [!UICONTROL 홈 달력]를 참조하십시오.  [[!UICONTROL 홈 달력] 보기](../../../workfront-basics/using-home/using-the-home-area/home-calendar-view.md).

1. 다음 중 하나를 수행합니다.

   * **작업 항목을 복수 일로 예약하려면** 2단계를 반복하여 작업할 각 일 및 시간에 항목을 추가합니다.
   * **달력 항목의 기간을 수정하려면 다음과 같이 하십시오.** 항목의 아래쪽 가장자리를 마우스로 가리키면 커서가 양방향 화살표가 될 때까지 아래 가장자리를 원하는 끝 시간으로 끕니다.
   * **달력 항목을 삭제하려면** 항목을 마우스로 가리킨 다음, [!UICONTROL 쓰레기통] 표시되는 아이콘입니다.

## 에서 작업 항목 업데이트 [!UICONTROL 홈 달력]

1. 로 이동합니다. [!UICONTROL 홈 달력] 에 설명된 대로 보기 [보기 [!UICONTROL 홈 달력]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).

1. 를 엽니다. [!UICONTROL 세부 사항] 업데이트할 작업 항목에 대한 패널:

   * 달력에서 예약된 달력 이벤트를 클릭합니다.

   작업 항목의 세부 사항이 창 오른쪽에 표시되는 패널에 표시됩니다.

1. 로 이동합니다. [!UICONTROL 홈 달력] 에 설명된 대로 보기 [보기 [!UICONTROL 홈 달력]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).
1. 클릭 **[!UICONTROL 일정 새로 고침]** 오른쪽 위 모서리에서 [!UICONTROL 홈 달력].

   ![](assets/refresh-qs-350x360.png)

   통합 달력 이벤트는 [!UICONTROL 홈 달력] 또는 를 새로 고칠 때 [!UICONTROL 홈 달력].
