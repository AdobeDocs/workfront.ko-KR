# [!DNL Android]에 대한 [!DNL Adobe Workfront]

다음 [!DNL Adobe Workfront] 앱을 사용하면 다음 항목에 대한 작업에 액세스할 수 있습니다. [!DNL Android] 디바이스. 다음을 설치하고 사용할 수 있습니다. [!DNL Workfront] 휴대 전화 및 태블릿의 모바일 앱 실행 중 [!DNL Android] 5.0 이상

모바일 앱에 로그인하는 방법에 대한 자세한 내용은 섹션 을 참조하십시오 [에 로그인합니다 [!DNL Workfront] 모바일 앱](../../../workfront-basics/manage-your-account-and-profile/managing-your-workfront-account/log-in-to-workfront.md#log) 이 문서에서 [에 로그인 [!DNL Adobe Workfront]](../../../workfront-basics/manage-your-account-and-profile/managing-your-workfront-account/log-in-to-workfront.md).

     <li>작업 및 문제 재할당</li>

    </ul> </td>

 </tbody>
