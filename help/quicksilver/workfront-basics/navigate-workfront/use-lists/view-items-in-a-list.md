# 에서 목록 시작 [!DNL Adobe Workfront]

<!--
{{highlighted-preview}}
-->

에서 객체 목록을 볼 수 있습니다 [!DNL Adobe Workfront] 시작 및 기한, 할당된 사용자 및 이와 연관된 기타 객체와 같은 정보를 얻기 위해

다음은 의 목록에 대한 몇 가지 특성입니다 [!DNL Workfront]:

* 5분마다 자동으로 새로 고침을 목록화하여 시스템의 다른 사용자가 다른 곳에서 업데이트할 정보를 업데이트합니다.
* 의 일부 영역 [!DNL Workfront] 는 기본 객체 목록으로 미리 구성되어 있습니다.

   이러한 사전 구성된 목록 대부분을 사용자 지정할 수 있습니다.

* A [!DNL Workfront] 관리자는 사용자 정의 목록을 만들어 다양한 영역에 적용할 수 있습니다 [!DNL Workfront].

   시스템 수준 목록 만들기에 대한 자세한 내용은 문서를 참조하십시오 [기본 필터, 보기 및 그룹 만들기, 편집 및 공유](../../../administration-and-setup/set-up-workfront/configure-system-defaults/create-and-share-default-fvgs.md).

![](assets/nwe-list-elements.png)

다음 영역에서 목록 요소를 사용자 지정하고 다른 사용자와 공유할 수 있습니다.

* 섹션에 있는 모든 시스템 기본 목록 [에서 목록 시작 [!DNL Adobe Workfront]](#default-workfront-lists) 이 문서
* 사용자와 공유되는 모든 보고서

목록의 빌드 요소는 보고서의 빌드 요소와 동일합니다.

목록 및 보고서의 기본 요소를 만들고 사용자 지정하는 방법에 대한 자세한 내용은 다음을 참조하십시오 [보고 요소: 필터, 보기 및 그룹화](../../../reports-and-dashboards/reports/reporting-elements/reporting-elements-filters-views-groupings.md).
