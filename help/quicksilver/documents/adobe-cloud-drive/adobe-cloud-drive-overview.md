# Adobe Cloud Drive 개요

Adobe Cloud Drive는 Adobe 클라우드 저장소 프로젝트를 Mac 또는 Windows 컴퓨터의 드라이브로 마운트하는 데스크탑 애플리케이션입니다. Workfront 프로젝트는 폴더로 표시되며, 사용자는 데스크탑 애플리케이션을 사용하여 Adobe 클라우드 스토리지에서 Workfront 프로젝트를 직접 열고, 만들고, 편집하고, 저장할 수 있습니다.

Adobe Cloud Drive의 파일은 Workfront 및 Frame.io에 표시되는 것과 동일한 파일이므로, 바탕 화면에서 수행하는 변경 사항은 두 제품의 연결된 Workfront 프로젝트에서 사용할 수 있습니다.

[Adobe Cloud Drive에서 사용할 수 있는 기능에 대한 비디오 개요를 봅니다.](https://video.tv.adobe.com/v/3492492){target="_blank"}

Adobe Cloud Drive가 연결되는 저장소에 대한 자세한 내용은 [Adobe 클라우드 저장소 개요](/help/quicksilver/review-and-approve-work/esm-overview.md)를 참조하세요.

## Adobe Cloud Drive로 수행할 수 있는 작업

* Adobe 클라우드 스토리지에서 Finder(Mac) 또는 파일 탐색기(Windows)의 폴더로 프로젝트에 액세스합니다.
* Adobe Creative Cloud 앱, Microsoft Office 또는 컴퓨터에 설치된 모든 애플리케이션에서 파일을 열고, 만들고, 편집합니다.
* 변경 사항을 로컬에 저장하고 Adobe Cloud Drive가 변경 사항을 Adobe 클라우드 저장소의 프로젝트와 동기화하도록 합니다.
* 네트워크 연결 없이 여행하거나 작업할 때 오프라인 액세스용으로 파일을 표시합니다. 연결이 복원되면 업데이트가 프로젝트의 파일과 동기화되고 로컬 캐시에서 파일을 제거하여 디스크 공간을 확보할 수 있습니다.
* 동일한 프로젝트 파일에서 팀 구성원과 공동 작업을 수행합니다.

## Adobe Cloud Drive 사용 가능 사용자

Adobe Cloud Drive를 사용하려면 귀사가 Workflow Ultimate 패키지에 있어야 하며 Adobe 클라우드 스토리지가 활성화되어 있어야 합니다.

Adobe 클라우드 저장소를 사용하는 프로젝트만 드라이브에 표시됩니다. 기존 Workfront 프로젝트가 표시되지 않습니다. 차이점에 대한 자세한 내용은 [Adobe 클라우드 저장소와 레거시 Workfront 저장소 간의 차이점](/help/quicksilver/review-and-approve-work/esm-overview.md#differences-between-adobe-cloud-storage-and-legacy-workfront-storage)을 참조하십시오.

Adobe Cloud Drive를 다운로드할 수 없는 경우나 조직에서 애플리케이션 설치를 관리하는 경우 Workfront 관리자에게 문의하십시오.
