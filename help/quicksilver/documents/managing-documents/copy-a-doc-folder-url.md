# 문서 폴더에 대한 링크 복사 및 공유

다음 중 하나에 포함된 문서 폴더에 대한 직접 링크를 복사할 수 있습니다 [!DNL Workfront] 개체: [!UICONTROL 프로그램], [!UICONTROL Portfolio], [!UICONTROL 프로젝트], [!UICONTROL 작업], 또는 [!UICONTROL 문제]. 에 포함된 폴더에 대한 링크를 복사할 수 없습니다 [!UICONTROL 문서] 영역을 클릭합니다. 이러한 폴더는 사용자 프로필에 직접 연결되어 있으므로 다른 사용자와 공유할 수 없습니다.

## 문서 폴더에 대한 링크 복사 및 공유

문서 폴더에 대한 링크를 복사하려면

1. 로 이동합니다 [!DNL Workfront] 문서 폴더를 포함하는 객체입니다.
1. 을(를) 클릭합니다. **[!UICONTROL 자세히]** 메뉴를 선택한 다음 **[!UICONTROL URL 복사]**. 이 링크를 공유하여 폴더에 빠르게 액세스할 수 있습니다. 폴더를 보려면 객체에 대한 보기 액세스 권한이 있어야 합니다.
   ![문서 폴더 url 복사](assets/copy-doc-folder-url.png)
