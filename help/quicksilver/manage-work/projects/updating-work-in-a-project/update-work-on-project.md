# 프로젝트에서 작업 업데이트

프로젝트의 상태와 진행 상태를 업데이트하려면 해당 작업의 상태와 진행 상태를 업데이트해야 합니다. 그것은 결국 그 프로젝트에 대한 진행 상황을 보여주기 위해 일어납니다.
