# 요청 만들기

<!--
{{highlighted-preview}}
-->

Adobe Workfront에서는 요청 및 문제를 서로 바꿔서 사용할 수 있습니다. 다음 중 하나를 수행하여 요청을 만들 수 있습니다.

* Workfront에서 요청 제출
* 다른 응용 프로그램에서 요청 제출
* 요청 큐로 설정된 프로젝트에 요청을 이메일로 보냅니다.

모든 요청은 제출된 프로젝트의 문제 영역에 표시됩니다.

에서 Workfront에 요청을 만들거나, 제출하거나, 이메일로 보내는 방법에 대한 자세한 내용은 다음 문서를 참조하십시오.

* [사용자가 요청 큐 프로젝트에 문제를 이메일로 보낼 수 있도록 설정](../../../manage-work/requests/create-requests/enable-email-issues-into-projects.md)
* [제출된 요청 찾기](../../../manage-work/requests/create-requests/locate-submitted-requests.md)
* [요청 큐에 대한 링크 공유](../../../manage-work/requests/create-requests/share-link-to-request-queue.md)
