## 작업에 대한 추적 모드 설정

1. 추적 모드를 설정할 작업으로 이동합니다.
1. 작업 이름 옆에 있는 **자세히** 아이콘 ![개체의 자세히 아이콘](assets/qs-more-icon-on-an-object.png)을 클릭한 다음 **편집**&#x200B;을 클릭합니다.

   작업 편집(Edit Task) 대화상자가 열립니다.

1. **설정** 섹션에서 **추적 모드** 드롭다운 메뉴를 사용하여 작업에 대한 추적 모드를 선택하십시오.

   ![작업 설정 편집 섹션의 추적 모드 필드](assets/tracking-mode-field-in-edit-task-settings-nwe-350x233.png)

1. 다음 옵션 중에서 선택합니다.

   * 사용자 업데이트 필요(기본 옵션)
   * 정시 추정
   * 지연 경고 무시
   * 자동 완성
   * 전임 작업

   추적 모드 옵션에 대한 자세한 내용은 [작업 추적 모드 개요](../../../manage-work/tasks/task-information/task-tracking-mode.md)를 참조하십시오.

1. **저장**&#x200B;을 클릭합니다.
