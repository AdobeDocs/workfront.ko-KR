# 작업 비용 유형 업데이트

태스크의 계획 및 실제 비용과 해당 노무비는 각 태스크의 원가 유형에 따라 결정됩니다.

프로젝트 내의 개별 작업에 대한 비용 유형을 구성할 수 있습니다. 각 원가 유형은 계획된 원가 및 실제 원가 값에 영향을 줍니다.

Adobe Workfront의 비용 추적에 대한 자세한 내용은 [비용 추적](../../../manage-work/projects/project-finances/track-costs.md)을 참조하세요.

## 개별 작업의 비용 유형 구성

1. 비용 유형을 구성할 작업으로 이동합니다.
1. 왼쪽 패널에서 **작업 세부 정보**&#x200B;를 클릭한 다음 **재무** 영역을 확장합니다.
1. **비용 유형**&#x200B;을(를) 두 번 클릭하고 작업에 적용할 비용 유형을 선택합니다.

   ![](assets/task-cost-type-options-in-tasks-details-nwe-350x233.png)

   다음 옵션 중에서 선택합니다.

   * 비용 없음
   * 고정 시간별
   * 시간별 사용자
   * 시간별 역할

   각 작업 비용 유형에 대한 자세한 내용은 [비용 추적](../../../manage-work/projects/project-finances/track-costs.md)을 참조하십시오.

1. **저장** **변경 내용** **.**&#x200B;을 클릭합니다.
