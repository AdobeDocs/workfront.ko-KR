# 스마트 할당 만들기

스마트 지정을 사용하여 작업을 완료하는 데 가장 적합한 사용자를 식별할 수 있습니다. 스마트 할당은 작업에 가장 적합한 리소스를 결정하는 알고리즘을 기반으로 리소스에 작업 항목을 할당할 때 Adobe Workfront이 사용자에게 표시하는 사용자에게 제안입니다. 스마트 할당에 대한 자세한 내용은 [스마트 할당 개요](../../../manage-work/tasks/assign-tasks/smart-assignments.md).

## 스마트 할당 만들기

스마트 배정은 Workfront에서 할당할 수 있는 대부분의 위치에서 사용할 수 있습니다.

1. 다음 영역으로 이동하여 **지정** 또는 **여기에 지정** 필드:   

   * 작업 또는 문제 목록 또는 보고서
   * 작업 또는 문제 헤더
   * 작업 또는 문제 요약 패널
   * 홈 영역에 나열된 항목에 대한 지정 필드
   * 예약 또는 예약 영역의 작업 또는 문제
   * 작업 로드 밸런서의 작업 또는 문제

1. 지정 필드에 커서를 놓고 2초 동안 기다립니다. ******제안된 지정** 목록이 표시됩니다.

   ![](assets/smart-assignments-task-header-nwe-350x302.png)

   이 목록에 표시되는 사용자는 작업 또는 문제에 대한 스마트 할당 제안 사항입니다.

   스마트 할당이 표시되는 위치에 대한 자세한 내용은 문서에서 &quot;스마트 할당 제안 찾기&quot; 섹션을 참조하십시오 [스마트 할당 개요](../../../manage-work/tasks/assign-tasks/smart-assignments.md) .

1. 해당 이름을 클릭하여 권장 사항 목록에서 사용자를 선택합니다. 클릭 **나에게 할당** 작업 항목을 직접 할당합니다.

   >[!TIP]
   >
   >제안이 없으면 제안 목록이 열리지 않습니다.

1. (선택 사항) 스마트 할당 목록에서 권장 사용자 중 하나를 사용하지 않으려면 원하는 사용자의 이름을 입력하고 목록에 표시될 때 해당 이름을 선택합니다.
1. 클릭 **Enter 키** 할당하려면

   선택한 사용자가 작업이나 문제에 할당됩니다.
