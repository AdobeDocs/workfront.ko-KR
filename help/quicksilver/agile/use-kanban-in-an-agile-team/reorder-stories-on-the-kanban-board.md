# [!UICONTROL Kanban] 보드에서 스토리 순서 바꾸기

[!UICONTROL Kanban] 보드의 상태 열에 스토리가 표시되는 순서를 변경할 수 있습니다.

>[!NOTE]
>
>[!UICONTROL Kanban] 팀의 백로그에 스토리가 표시되는 순서를 변경할 수 없습니다.

[!UICONTROL Kanban] 보드에 스토리가 표시되는 순서는 우선 순위를 나타내지 않습니다. 다만, 이야기를 더 잘 보이게 함으로써 인지된 우선순위에 영향을 줄 수 있다.

기본적으로 스토리는 [!UICONTROL Kanban] 보드의 각 상태 열 내에 알파벳 순서로 표시됩니다.

[!DNL Kanban] 보드에서 열 순서를 변경하면 변경한 내용이 모두 저장되므로 다음 번에 사용자나 다른 사용자가 [!UICONTROL Kanban] 보드를 볼 때 변경 내용이 유지됩니다. (브라우저 캐시를 지울 때 수행한 변경 사항은 되돌릴 수 없습니다.)

## [!UICONTROL Kanban] 보드에서 스토리 순서 바꾸기

{{step1-to-team}}

1. (선택 사항) **[!UICONTROL 팀 전환]** 아이콘 ![팀 전환 아이콘](assets/switch-team-icon.png)을 클릭한 다음 드롭다운 메뉴에서 새 [!UICONTROL Kanban] 팀을 선택하거나 검색 창에서 팀을 검색합니다.

1. 순서를 변경할 스토리가 포함된 [!UICONTROL Kanban] 보드로 이동합니다.
1. [!UICONTROL Kanban] 보드의 상태 열 내에서 원하는 세로 위치로 스토리 타일을 끌어옵니다.
