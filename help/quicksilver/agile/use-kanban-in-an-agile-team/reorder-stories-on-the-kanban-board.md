# 에서 스토리 순서 조정 [!UICONTROL 간판] 보드

스토리의 상태 열 내에 스토리가 표시되는 순서를 [!UICONTROL 간판] 보드.

>[!NOTE]
>
>다음에 대한 백로그에 스토리가 표시되는 순서를 변경할 수 없습니다 [!UICONTROL 간판] 팀

스토리가 표시되는 순서 [!UICONTROL 간판] 보드는 우선 순위를 나타내지 않습니다. 하지만, 이것은 이야기를 더 잘 보이도록 함으로써 인식된 우선순위에 영향을 줄 수 있습니다.

기본적으로 스토리는 [!UICONTROL 간판] 보드.

에서 열 순서를 변경할 때 [!DNL Kanban] 보드, 사용자가 변경한 모든 내용이 저장되므로 다음 번에 사용자가 보거나 다른 사용자가 볼 때 변경 사항이 유지됩니다 [!UICONTROL 간판] 보드. (수행하는 변경 사항은 브라우저 캐시를 지울 때 복귀되지 않습니다.)

## 에서 스토리 순서 조정 [!UICONTROL 간판] 보드

순서 또는 스토리를 변경하려면:

1. 을(를) 클릭합니다. **[!UICONTROL 기본 메뉴]** 아이콘 ![](assets/main-menu-icon.png) 의 오른쪽 위 모서리에서 [!DNL Adobe] Workfront을 클릭한 다음 **[!UICONTROL 팀]**.

1. (선택 사항) **[!UICONTROL 팀 전환]** 아이콘 ![팀 전환 아이콘](assets/switch-team-icon.png)를 선택한 다음, 새로 만들기를 선택합니다 [!UICONTROL 간판] 드롭다운 메뉴에서 팀이나 검색 막대에서 팀을 검색합니다.

1. 로 이동합니다. [!UICONTROL 간판] 재정렬하려는 스토리가 포함된 게시판입니다.
1. 스토리 타일을 의 상태 열 내의 원하는 세로 위치로 드래그합니다. [!UICONTROL 간판] 보드.
