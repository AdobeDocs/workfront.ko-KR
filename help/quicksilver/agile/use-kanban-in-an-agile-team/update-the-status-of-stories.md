# 에서 스토리 상태 업데이트 [!UICONTROL 간판] 보드

스토리의 상태를 [!UICONTROL 간판] 이야기가 어떻게 진행되는지 반영하기 위해 게시하다.

>[!NOTE]
>
>에서 선택한 상태만 [!UICONTROL 스토리 보드] 의 섹션 [!UICONTROL 팀 설정] 영역은 [!UICONTROL 간판] 상태 드롭다운 메뉴에서 및 를 포함합니다. 자세한 내용은 [간판 구성](../../agile/get-started-with-agile-in-workfront/configure-kanban.md)

## 간판 보드의 스토리 상태 업데이트

1. 을(를) 클릭합니다. **[!UICONTROL 기본 메뉴]** 아이콘 ![](assets/main-menu-icon.png) 의 오른쪽 위 모서리에서 [!DNL Adobe Workfront]를 클릭한 다음 **[!UICONTROL 팀]**.

1. (선택 사항) **[!UICONTROL 팀 전환]** 아이콘 ![팀 전환 아이콘](assets/switch-team-icon.png)를 선택한 다음, 새로 만들기를 선택합니다 [!UICONTROL 간판] 드롭다운 메뉴에서 팀이나 검색 막대에서 팀을 검색합니다.

1. 로 이동합니다. [!UICONTROL 간판] 스토리의 상태를 업데이트할 보드입니다.
1. 한 상태 열의 스토리를 [!UICONTROL 간판] 및 를 다른 열로 전환합니다.\
   이야기는 그대로 남아 있다 [!UICONTROL 완료] 열이 추가된 후 2주 동안 열이 비어 있습니다.\
   ![스토리를 다른 열로 이동](assets/agile-storyboard-progress.png)
