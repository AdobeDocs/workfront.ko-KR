# Web Proofing Viewer와 Desktop Proofing Viewer 개요의 차이점

Adobe Workfront은 두 가지 서로 다른 언어 교정 뷰어를 제공합니다.

* **웹 언어 교정 뷰어:** 주로 정적 및 비디오 파일 교정을 위해 설계되었습니다. Google Chrome, Firefox 또는 Safari에서 실행됩니다.
* **데스크탑 언어 교정 뷰어:** 비디오 및 정적 파일은 물론 대화형 파일도 교정할 수 있도록 설계되었습니다. 워크스테이션에서 독립 실행형 애플리케이션으로 실행됩니다. 자세한 내용은 [데스크탑 교정 뷰어 이해](../../../workfront-proof/wp-work-proofsfiles/review-proofs-dpv/destop-proofing-viewer.md)

* 보안상의 이유로 조직에서 데스크탑 언어 교정 뷰어 앱을 사용할 수 없는 경우 Workfront 관리자가 웹 언어 교정 뷰어에서 ZIP 아카이브 파일에 번들로 제공된 대화형 컨텐츠를 검토할 수 있도록 시스템을 구성할 수 있습니다. 자세한 내용은  [Web Proofing Viewer에서 대화형 콘텐츠 교정 설정](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/interactive-content-in-web-viewer.md).

다음 목록을 통해 특정 유형의 콘텐츠를 입증하는 데 사용할 수 있는 언어 교정 뷰어를 이해할 수 있습니다.

* **대화형 웹 컨텐츠 - URL**: URL을 사용하여 웹 컨텐츠에 대한 증명을 만들고 내용을 대화식으로 증명하려면 데스크탑 언어 교정 뷰어를 사용해야 합니다.
* **대화형 웹 컨텐츠 - ZIP 파일**: ZIP 파일을 사용하여 웹 컨텐츠에 대한 증명을 만들 경우 웹 언어 교정 뷰어(일부 제한 사항 있음) 또는 데스크탑 언어 교정 뷰어를 사용할 수 있습니다. 대화형 컨텐츠에 Web Proofing Viewer를 사용하는 것과 관련된 제한 사항에 대한 자세한 내용은 [Web Proofing Viewer에서 대화형 콘텐츠 교정 설정](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/interactive-content-in-web-viewer.md).

* **정적 및 비디오 컨텐츠**: 정적 콘텐츠가 들어 있는 증명을 만들 경우 Web Proofing Viewer 또는 Desktop Proofing Viewer를 사용할 수 있습니다.
