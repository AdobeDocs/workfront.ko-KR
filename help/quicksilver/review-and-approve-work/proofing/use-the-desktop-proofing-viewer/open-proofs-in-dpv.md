## 전제 조건

* Desktop Proofing Viewer를 설치해야 합니다. 자세한 내용은 [데스크탑 증명 뷰어 설치](../../../review-and-approve-work/proofing/use-the-desktop-proofing-viewer/installing-desktop-proofing-viewer.md)를 참조하십시오.

## 데스크탑 교정 뷰어에서 증명을 엽니다.

데스크탑 언어 교정 뷰어에서 대화형 내용을 정기적으로 검토하는 경우, Workfront 관리자는 데스크탑 언어 교정 뷰어를 기본 뷰어로 설정할 수 있습니다. 자세한 내용은 [데스크탑 교정 뷰어에서 비대화형 증명을 여는 사용자 설정](../../../workfront-proof/wp-work-proofsfiles/review-proofs-dpv/destop-proofing-viewer.md#user-setting-for-launching-non-interactive-proofs) 기사에서는 [데스크탑 교정 뷰어 이해](../../../workfront-proof/wp-work-proofsfiles/review-proofs-dpv/destop-proofing-viewer.md).

1. 데스크탑 언어 교정 뷰어에서 **증명 URL**.

   증명 URL에 액세스하는 방법에 대한 내용은 [공유된 증명 액세스](../../../workfront-proof/wp-work-proofsfiles/share-proofs-and-files/access-proofs-shared-with-you.md).

   >[!NOTE]
   >
   >데스크톱 언어 교정 뷰어는 로그인해야 하는 웹 사이트를 지원하지 않습니다.

1. 클릭 **공개 증명** 표시됩니다.
