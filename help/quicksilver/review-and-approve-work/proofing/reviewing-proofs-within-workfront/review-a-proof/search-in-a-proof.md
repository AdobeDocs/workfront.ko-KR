# 증명 내 콘텐츠 검색

다음 유형의 문서에 대해 작성된 증명에서 텍스트를 빠르게 찾을 수 있습니다.

* PDF
* Office(.doc, .docx, .odt)
* 정적 웹 페이지

>[!NOTE]
>
>2017년 4월 26일 이전에 생성된 증명은 검색할 수 없습니다.

## 증명 내 콘텐츠 검색

1. 검색할 증명을 엽니다.
1. 증명 위의 도구 모음에서 **문서 검색** 아이콘.

   ![](assets/search-document-icon-search-in-proof-350x129.png)

1. 검색할 텍스트를 입력하십시오.

   검색 도구는 사용자가 입력하는 동안 문서의 텍스트를 강조 표시합니다.

   ![Search_text_highlighted_as_you_type.png](assets/search-text-highlighted-as-you-type-350x214.png)

1. 검색할 텍스트 입력을 마친 다음 **위로** 및 **아래로** 증명 내의 검색 결과를 검색하는 화살표
