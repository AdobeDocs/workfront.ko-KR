# 요약을 사용하여 작업 로드 밸런서에서 작업 항목 업데이트

작업 또는 문제의 요약 패널을 사용하여 해당 항목에 대한 중요한 정보를 업데이트할 수 있습니다. 작업 로드 밸런서에서 다음을 수행합니다.

요약 패널은 작업 로드 밸런서의 지정된 작업 영역 및 지정되지 않은 작업 영역 모두에서 작업 및 문제에 사용할 수 있습니다.

## 작업 로드 밸런서 요약 패널에서 작업 및 문제 업데이트

1. 작업 로드 밸런서로 이동합니다.

   작업 로드 밸런서는 현재 주부터 시작되는 작업 지정 정보를 표시합니다.

1. 로 이동합니다. **할당되지 않은 작업** 영역을 설정하고 작업 항목을 찾습니다.

   또는

   로 이동합니다. **지정된 작업 시간** 영역을 확장하고 사용자를 확장하고 작업 항목을 찾습니다.

1. 작업 또는 문제 표시줄을 클릭하여 오른쪽의 요약 패널을 엽니다

   또는

   클릭 **요약 열기** 아이콘 ![](assets/summary-panel-icon.png)를 클릭한 다음 작업 또는 문제 표시줄을 클릭하여 요약 패널을 엽니다

   또는

   을(를) 클릭합니다. **자세히** 메뉴 ![](assets/more-icon.png) 작업 또는 문제 오른쪽의 **요약 열기**.

   요약 패널이 오른쪽에 열립니다.

   ![](assets/summary-panel-for-task-in-workload-balancer-350x449.png)

1. 작업 항목에 대한 필드 수를 수정하거나 업데이트를 입력하거나 여기에서 Workfront의 다른 영역으로 이동하여 문서 또는 기타 정보를 작업 항목에 추가합니다.

   >[!CAUTION]
   >
   >[요약]에서 [하위 작업] 섹션으로 이동하여 하위 작업을 추가하면 상위 작업이 작업 로드 밸런서에 표시되지 않으므로 원래 작업이 제거됩니다.

   요약을 사용하여 정보를 업데이트하는 방법에 대한 자세한 내용은 [요약 개요](../../workfront-basics/the-new-workfront-experience/summary-overview.md).

1. (조건부) 작업 항목의 섹션을 연 경우 브라우저의 **뒤로 단추** 작업 로드 밸런서로 돌아가려면
1. 을(를) 클릭합니다. **닫기 아이콘** ![](assets/close-icon.png) 요약 오른쪽 상단 모서리에서 을(를) 클릭하여 닫습니다.
