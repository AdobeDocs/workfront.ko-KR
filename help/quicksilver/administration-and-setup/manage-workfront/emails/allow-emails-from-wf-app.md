# Adobe Workfront 애플리케이션에서 이메일 허용

특정 공급업체에 대한 액세스만 허용하도록 방화벽 또는 메일 서버를 구성한 경우 특정 Adobe Workfront IP 주소를 방화벽 IP에 추가해야 합니다. 지침은 [방화벽 구성](../../../administration-and-setup/get-started-wf-administration/configure-your-firewall.md)을 참조하세요.
