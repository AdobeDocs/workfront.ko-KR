# 시스템 프로젝트 상태

Workfront에는 9개의 내장된 시스템 프로젝트 상태가 있습니다.

아래 표의 첫 번째 3이 필요합니다. 즉, 잠금을 해제하고 이름을 변경하고 순서를 변경할 수 있지만 숨기거나 삭제할 수는 없습니다.

프로젝트 상태를 변경하는 것은 일반적으로 수동 프로세스입니다. 하지만 시스템에서 발생하는 다른 요소에 따라 프로젝트 상태가 자동으로 변경될 때 다음 목록에 요약된 몇 가지 시나리오가 있습니다.

다음 프로젝트 상태가 Workfront 인스턴스와 함께 제공됩니다.
