# 조건 표시 순서 변경

프로젝트, 작업 및 문제에 조건이 표시되는 순서를 변경할 수 있습니다.

* 사용자가 프로젝트를 편집하는 경우

   ![](assets/change-condition-edit-project.png)

* 사용자가 작업 또는 문제에 대한 조건을 변경하는 경우

   ![](assets/change-conditions-list-dropdown-only.png)

## 조건 표시 순서 변경

1. 을(를) 클릭합니다. **기본 메뉴** 아이콘 ![](assets/main-menu-icon.png) Adobe Workfront의 오른쪽 위 모서리에서 을(를) 클릭하고 **설정** ![](assets/gear-icon-settings.png).

1. 클릭 **프로젝트 환경 설정** > **조건**.

1. 을(를) 선택합니다 **프로젝트**,**작업**, 또는 **문제** 탭.

1. 드래그 ![](assets/move-icon---dots.png) 주문 변경 조건입니다.

   새 주문은 자동으로 저장됩니다.

조건 사용자 지정에 대한 자세한 내용은 [사용자 지정 조건](../../../administration-and-setup/customize-workfront/create-manage-custom-conditions/custom-conditions.md).
