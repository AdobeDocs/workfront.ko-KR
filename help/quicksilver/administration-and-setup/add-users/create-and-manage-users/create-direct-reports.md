# 직접 보고서 만들기

사용자를 다른 사용자에게 직접 보고서로 지정할 수 있습니다. 조직의 관리 구조를 표시하는 조직 차트를 만들 수 있습니다. 자세한 내용은 [조직도 보기](../../../people-teams-and-groups/work-directly-with-others/view-the-org-chart.md).

## 직접 보고서 만들기

1. 에 설명된 대로 사용자 편집을 시작합니다. [사용자 프로필 편집](../../../administration-and-setup/add-users/create-and-manage-users/edit-a-users-profile.md).
1. 에서 **조직** 섹션에서 **회사** 필드.

   이 필드는 비워 둘 수 없습니다.

1. 에서 **직접 보고서** 필드에서, 편집 중인 사용자에게 보고하는 사용자를 지정합니다.

1. (선택 사항)에서 **보고서 대상** 필드에서 이 사용자가 보고하는 사용자의 이름을 지정합니다.
1. 클릭 **변경 내용 저장**.
