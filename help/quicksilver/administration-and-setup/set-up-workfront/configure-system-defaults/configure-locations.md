# 위치 구성

{{highlighted-preview-article-level}}

요금 카드를 사용하면 조직에서 프로젝트의 청구 요율을 쉽게 관리할 수 있습니다. 자세한 내용은 [등급 카드 관리](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/manage-rate-cards.md). (으)로 [!DNL Adobe] [!DNL Workfront] 관리자는 비율 카드의 작업 역할에 할당할 수 있는 기본 위치를 구성할 수 있습니다. 이렇게 하면 요금 카드가 각 위치의 시장 요금을 정확하게 반영합니다.

## 전체 조직을 위한 위치 구성

{{step-1-to-setup}}

1. 왼쪽 패널에서 [!UICONTROL **위치**].
1. 클릭 [!UICONTROL **더 많은 위치 추가**].
1. 위치 이름과 설명을 입력합니다.

   위치 이름은 대문자로 시작해야 하며 위치를 복제할 수 없습니다.

1. 시작 영역 외부를 클릭하여 위치를 저장합니다.
1. 위치를 삭제하려면 목록에서 위치를 선택하고 **삭제** 아이콘 ![삭제 아이콘](assets/delete.png).

>[!NOTE]
>
>비율 카드의 작업 역할과 연결된 위치는 삭제할 수 없습니다.
