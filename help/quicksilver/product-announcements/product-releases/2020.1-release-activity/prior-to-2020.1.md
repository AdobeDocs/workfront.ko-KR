# 2020.1 릴리스 전에 프로덕션에 릴리스된 다른 Workfront Classic 기능

다음 기능은 2020.1 프로덕션 릴리스 전에 Workfront Classic 프로덕션 환경에서 사용할 수 있었습니다.
