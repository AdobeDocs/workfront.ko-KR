# Adobe Workfront Planning의 2026년 2분기 릴리스 활동

이 문서에서는 2026년 2분기 릴리스 동안 Workfront Planning에 대해 릴리스되는 기능에 대해 설명합니다.

<!--keep the sentence below for all future quarterly release pages-->

Adobe Workfront Planning에 대해 릴리스된 모든 기능 목록을 보려면 [Adobe Workfront Planning 릴리스 활동: 문서 색인](/help/quicksilver/product-announcements/product-releases/planning-release-activity/planning-release-activity-article-index.md)을 참조하십시오.

## Workfront Planning의 사용자 자동 완성 필드에 이메일 표시

>[!NOTE]
>
>미리 보기: 2026년 1월 22일
>프로덕션 빠른 릴리스: 2026년 2월 12일
>모두를 위한 프로덕션: 2026년 4월 16일

이제 다음 영역에 사용자를 추가하면 사용자의 이메일이 표시됩니다.

* 사용자 유형 계획 필드
* Planning 객체를 공유할 때 공유 상자에서

이 개선 이전에는 이러한 영역에 사용자를 추가할 때 사용자 이름 외에 사용자의 기본 역할만 표시되었습니다.
자세한 내용은 다음 문서를 참조하십시오.

* [필드 만들기](/help/quicksilver/planning/fields/create-fields.md)
* [작업 공간 공유](/help/quicksilver/planning/access/share-workspaces.md)

## 사람 필드는 Planning 연결에 대한 조회 필드로 활성화되어 있습니다.

>[!NOTE]
>
>미리 보기: 2026년 1월 14일
>프로덕션 빠른 릴리스: 2026년 2월 12일
>모두를 위한 프로덕션: 2026년 4월 16일

이제 두 Planning 레코드 유형을 연결할 때 사람 필드를 조회 필드에 추가할 수 있습니다.

자세한 내용은 [레코드 종류 연결](/help/quicksilver/planning/architecture/connect-record-types.md)을 참조하세요.
