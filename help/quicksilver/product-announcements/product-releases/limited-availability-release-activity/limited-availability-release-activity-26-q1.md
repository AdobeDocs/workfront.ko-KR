# Adobe Workfront의 제한된 가용성 기능에 대한 2026년 1분기 릴리스 활동

이 문서에서는 2026년 1분기 중에 릴리스되는 제한된 가용성의 Workfront 기능에 대해 설명합니다.

Adobe Workfront에 대해 릴리스된 모든 제한된 가용성 기능의 목록을 보려면 [Adobe Workfront 제한된 가용성 기능 릴리스 활동: 문서 인덱스](/help/quicksilver/product-announcements/product-releases/limited-availability-release-activity/limited-availability-release-activity-article-index.md)을 참조하십시오.

## 제한된 가용성 기능에 대한 릴리스 정보

이 섹션에는 새로운 기능에 대한 릴리스 정보가 포함되어 있습니다.

### 비율 카드의 작업 역할 비율 조정

>[!NOTE]
>
>프로덕션 릴리스 날짜: 2025년 11월 20일

이제 요금 카드 내에서 하나 이상의 작업 역할 청구 요금에 백분율 기반 변경 사항을 적용할 수 있습니다. 각 비율을 하나씩 수동으로 업데이트하는 대신 한 번의 작업으로 선택한 모든 비율에 변경 사항을 적용할 수 있습니다.

## 제한된 가용성 기능에 대한 유지 보수 업데이트

이 섹션에서는 제한된 가용성 기능에 대한 주간 Workfront 업데이트에서 해결된 문제를 설명합니다.

### 2025년 11월 23~29일 유지 보수 업데이트 주

**고급 할당에 대한 고급 검색에서 시간/FTE/백분율 제거됨**

고급 할당 고급 검색에서 **시간/FTE/백분율**&#x200B;에 대한 옵션을 선택해도 검색 표시가 변경되지 않았습니다. 드롭다운이 도구 모음에서 제거되었습니다.

작업 기간이 0 **인 경우**&#x200B;고급 할당에서 계획된 시간을 편집할 수 없습니다.

작업 기간이 0인 경우 고급 할당 화면의 계획된 시간 KPI를 더 이상 편집할 수 없습니다. 이전에는 계획된 시간을 편집할 수 있었지만 변경 사항이 저장되지 않았습니다.

### 2025년 11월 30일부터 12월 6일의 유지 보수 업데이트 주

**환율 유효 환율을 사용하지 않을 때 환율 일자 필드가 숨겨짐**

**시스템에서 환율 사용** 확인란을 선택하지 않으면 프로젝트 세부 정보 > 재무 영역의 **환율 날짜** 필드가 이제 숨겨집니다. 이전에는 프로젝트를 편집할 때 날짜 유효 환율이 활성화되지 않은 경우에도 프로젝트 세부 정보에 날짜 필드가 표시되었습니다.
