# Adobe Workfront의 제한된 가용성 기능에 대한 2026년 2분기 릴리스 활동

이 문서에서는 2026년 2분기 중에 릴리스되는 제한된 가용성의 Workfront 기능에 대해 설명합니다.

Adobe Workfront에 대해 릴리스된 모든 제한된 가용성 기능의 목록을 보려면 [Adobe Workfront 제한된 가용성 기능 릴리스 활동: 문서 인덱스](/help/quicksilver/product-announcements/product-releases/limited-availability-release-activity/limited-availability-release-activity-article-index.md)을 참조하십시오.

## 제한된 가용성 기능에 대한 릴리스 정보

이 섹션에는 새로운 기능에 대한 릴리스 정보가 포함되어 있습니다.

### 작업 단일 또는 일괄 할당 시 경험 업데이트됨

>[!NOTE]
>
>프로덕션 릴리스 날짜: 2026년 2월 12일

단일 작업을 편집하거나 일괄 편집할 때 작업 편집 상자의 할당 섹션을 업데이트했습니다.

작업에 리소스를 할당하는 새로운 경험이 있습니다. 이제 하나의 작업을 편집하거나 여러 작업을 일괄적으로 편집할 때 모두 사용할 수 있습니다. 할당 섹션에 다음 필드를 추가했습니다.

* 할당
* 소유자 또는 작업 소유자
* 할당자의 역할

자세한 내용은 [작업 편집](/help/quicksilver/manage-work/tasks/manage-tasks/edit-tasks.md)을 참조하세요.
