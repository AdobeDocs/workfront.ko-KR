### Jira 통합을 위한 Workfront 사용을 위한 필수 IP 허용 목록 업데이트

Workfront for Jira 통합에 대한 백엔드 업데이트를 만들었습니다. [방화벽 구성](../../../administration-and-setup/get-started-wf-administration/configure-your-firewall.md)에 나열된 다음 IP 주소가 인바운드 및 아웃바운드 연결을 위한 허용 목록에 추가하다허용 목록에 추가하다 에 추가되었는지 확인하십시오.

* 34.213.36.118
* 35.160.0.242
* 3.209.27.146
* 18.205.251.4

이러한 주소는 이전에 Workfront의 다른 영역에 필요한 것으로 나열되었지만, 이제 Jira 통합에도 필요합니다.

Jira 통합을 위한 Workfront이 중단되는 것을 방지하려면 조직에서 2021년 8월 4일까지 이러한 IP 주소를 허용해야 합니다.

Jira 통합 구성에 대한 자세한 내용은 [Jira용 Adobe Workfront 설치](../../../workfront-integrations-and-apps/use-workfront-with-jira/install-workfront-for-jira.md)를 참조하십시오.

### API 버전 13

API 버전 13의 경우 다양한 리소스 및 엔드포인트를 수정했습니다. 일부 변경 사항은 새로운 기능을 지원하며, 다른 변경 사항을 통해 API를 통해 사용 가능한 정보를 보다 쉽게 사용할 수 있습니다.

새로운 기능 및 업데이트 내용은 [API 버전 13의 새로운 기능](../../../wf-api/api/new-api-version-13.md)을 참조하세요.
