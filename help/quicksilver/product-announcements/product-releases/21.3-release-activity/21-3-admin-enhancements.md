# 21.3 관리자 개선 사항

이 페이지에서는 미리 보기 환경에 대한 21.3 릴리스로 향상된 모든 관리자 기능을 설명합니다. 이러한 개선 사항은 2021년 7월 21일이 있는 프로덕션 환경에서 사용할 수 있었습니다.

21.3 릴리스에서 사용할 수 있는 모든 변경 사항 목록에 대해서는 다음을 참조하십시오 [21.3 릴리스 개요](../../../product-announcements/product-releases/21.3-release-activity/21-3-release-overview.md).

## 21.3 릴리스와 함께 프로덕션에서 사용 가능한 블루프린트

블루프린트 기능은 일반적으로 2021년 6월 21일이 있는 주의 21.3 프로덕션 릴리스에서 사용할 수 있습니다.

Blueprint는 기본 구성 요소를 제공하여 사용자 성장에 따라 성장하는 작업 관리 시스템을 만드는 데 도움이 됩니다. 시스템 관리자는 Blueprint 카탈로그를 탐색하고 사용 가능한 프로젝트 템플릿을 설치할 수 있습니다.

자세한 내용은 [블루프린트 개요](../../../administration-and-setup/blueprints/blueprints-overview.md).

이 기능은 이제 [새 Workfront 경험의 프로젝트 템플릿](https://one.workfront.com/s/learningpath4/project-templates-in-the-new-workfront-experience-MCGLS7GRNLDZDFPF6AEOGIDZFDG4) Workfront One에서 학습 경로.

## 그룹 관리자를 위한 새로운 기능: 그룹에 대한 상태를 임의의 수준에서 만들고 관리합니다

조직의 모든 수준에서 독립적으로 워크플로우를 관리하고 제어할 수 있도록 하위 그룹에 대한 상태를 만들고 관리하는 기능을 도입했습니다. 이제 설정의 그룹 섹션에서 모든 수준에서 관리하는 그룹에 대해 다음을 수행할 수 있습니다.

* 그룹의 상태 만들기, 편집, 삭제 및 숨기기
* 아래의 모든 하위 그룹이 동일한 방식으로 사용할 수 있도록 그룹에 대한 상태를 잠급니다
* 하위 그룹 관리자가 고유한 요구 사항에 맞게 사용자 지정할 수 있도록 모든 그룹의 상태를 잠금 해제하십시오
* 그룹 상태를 기본 상태로 설정
* 개체의 그룹 상태 표시 순서 변경 및 숨기기

Workfront 관리자는 모든 그룹에 대해 이러한 작업을 수행할 수도 있습니다.

이전에는 이 기능을 최상위 그룹에만 사용할 수 있었습니다.

자세한 내용은 [그룹 상태 관리](../../../administration-and-setup/manage-groups/manage-group-statuses/manage-group-statuses.md).

## 관리자를 위한 새로운 기능: 청구 레코드 사용자 지정 양식 만들기

이제 청구 레코드 사용자 지정 양식을 만들어 사용자가 청구 정보를 캡처하는 방법을 개선할 수 있습니다. 사용자는 사용자 지정 양식을 청구 레코드에 첨부하고, 양식을 작성하고, 양식에 정보가 포함된 보고서를 실행할 수 있습니다.

사용자 지정 양식 만들기에 대한 내용은 [사용자 지정 양식 만들기 또는 편집](../../../administration-and-setup/customize-workfront/create-manage-custom-forms/create-or-edit-a-custom-form.md).

프로젝트에 청구 레코드 사용자 지정 양식 첨부에 대한 자세한 내용은 [개체에 사용자 지정 양식 추가](../../../workfront-basics/work-with-custom-forms/add-a-custom-form-to-an-object.md).
