# 2025년 2분기 릴리스 기간 동안의 기타 개선 사항

이 페이지에서는 미리보기 환경에 대한 2025년 2분기 릴리스의 향상된 기능에 대해 설명합니다. 이러한 개선 사항은 명시된 대로 프로덕션 환경에서 사용할 수 있습니다.

2025년 2분기 릴리스 주기에 있는 이 시점에서 사용 가능한 모든 변경 내용의 목록은 [2025년 2분기 릴리스 개요](/help/quicksilver/product-announcements/product-releases/25-q2-release-activity/25-q2-release-overview.md)를 참조하십시오.

## 버전 업그레이드 끝점을 사용하는 새 이벤트 구독 버전으로 업그레이드

>[!NOTE]
>
>모든 고객을 위한 프로덕션: 2025년 3월 6일

이제 Workfront에는 이벤트 구독 버전이 있습니다. 새 버전은 Workfront API에 대한 변경이 아니라 이벤트 구독 기능에 대한 변경입니다.

이벤트 구독을 업그레이드하거나 다운그레이드할 수 있으므로 이벤트 구조를 변경할 때 기존 구독이 중단되지 않으므로 이벤트 구독에서 간격 없이 새 버전으로 테스트 및 업그레이드할 수 있습니다.

두 버전의 차이점에 대한 자세한 내용은 문서 [이벤트 구독 버전 관리](/help/quicksilver/wf-api/general/event-subs-versioning.md)를 참조하십시오.

버전 간 이벤트 구독을 업그레이드하거나 다운그레이드하는 데 사용되는 종단점에 대한 자세한 내용은 이벤트 구독 API 문서의 [이벤트 구독 버전 관리](/help/quicksilver/wf-api/general/event-subs-api.md#event-subscription-versioning) 섹션을 참조하십시오.

## Adobe Admin Console 업데이트 피드에 Workfront 사용자 변경 사항을 &quot;시스템&quot;으로 표시

>[!NOTE]
>
>미리 보기 릴리스: 2025년 1월 23일, 빠른 릴리스를 위한 프로덕션: 25.2 릴리스 포함(2025년 2월 13일), 분기별 릴리스를 위한 프로덕션: 25.4 릴리스 포함(2025년 4월)

이제 Adobe Admin Console 관리자가 Workfront 사용자의 사용자 정보를 변경하면 Workfront은 사용자 업데이트 영역의 시스템 활동 탭에 &quot;시스템&quot;에 속하는 것으로 이 변경 사항을 기록합니다. Adobe Admin Console 관리자를 말합니다.

이 업데이트 이전에는 Admin Console 관리자가 수행한 사용자 변경 사항이 주 Workfront 시스템 관리자가 수행한 대로 기록되었습니다.

Adobe Admin Console에서 사용자 관리에 대한 자세한 내용은 [Adobe Admin Console에서 사용자 관리](/help/quicksilver/administration-and-setup/add-users/create-and-manage-users/admin-console.md)를 참조하십시오.
