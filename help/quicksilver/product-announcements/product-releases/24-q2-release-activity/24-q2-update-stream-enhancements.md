# 2024년 2분기 업데이트 스트림 및 알림 개선 사항

이 페이지에서는 미리 보기 환경에 대한 2024년 2분기 릴리스의 모든 업데이트 스트림 및 알림 개선 사항에 대해 설명합니다. 향상된 기능은 앞서 설명한 대로 프로덕션 환경에서 사용할 수 있습니다.

2024년 2분기 릴리스 주기의 현재 사용 가능한 모든 변경 사항에 대한 목록은 [2024년 2분기 릴리스 개요](/help/quicksilver/product-announcements/product-releases/24-q2-release-activity/24-q2-release-overview.md)를 참조하십시오.

## 반복에 추가된 새 주석 달기 경험

>[!NOTE]
>
>미리보기 릴리스: 2024년 4월 11일, 모든 고객을 위한 프로덕션: 24.4 릴리스 포함(2024년 4월 11일)

응용 프로그램 전체에서 일관된 경험을 제공하기 위해 새로운 주석 달기 환경을 반복에 추가했습니다. 자세한 내용은 [섹션 개요 업데이트](/help/quicksilver/workfront-basics/updating-work-items-and-viewing-updates/updates-tab-overview.md)를 참조하세요.

## 레거시 주석 달기 영역에 대한 액세스 권한이 제거되었습니다.

>[!NOTE]
>
>Preview release: April 1, 2024; Production for all customers: With the 24.4 release (April 11, 2024)

We have removed the &quot;New commenting&quot; toggle in the Updates area for projects, tasks, issues, and documents. The new commenting is now the default and only option for these objects. With this change, you are no longer able to revert back to the legacy commenting experience. Additionally, we have removed the &quot;Show percent complete on update status&quot; setting on a user&#39;s profile.

For more information, see [New commenting experience](/help/quicksilver/product-announcements/betas/new-commenting-experience-beta/unified-commenting-experience.md).

## 이제 새 홈 위젯에 대한 빠른 작업에서 새 댓글 달기 스트림을 사용할 수 있습니다.

>[!NOTE]
>
>Preview release: March 1, 2024; Production for fast release: With the 24.3 release (March 14, 2024); Production for all customers: With the 24.4 release (April 11, 2024)

We have now added the new commenting functionality to the **Add new update** quick action button in the My Work, My Projects, My Tasks, and My Issues widgets in new Home.


## Additional commenter information available in the new commenting experience

>[!NOTE]
>
>Preview release: February 29, 2024; Production for fast release: With the 24.3 release (March 14, 2024); Production for all customers: With the 24.4 release (April 11, 2024)

We have made the following enhancements in the new commenting experience:

* You can click a commenter&#39;s name to see their name, role, and email address in an information box. 이 정보는 동일한 이름을 가진 사용자가 두 명 이상 있는 경우 올바른 사용자를 식별하는 데 도움이 될 수 있습니다. 정보 상자에서 댓글 작성자의 이름을 클릭하면 해당 댓글 작성자의 사용자 프로필이 열립니다.
* 댓글 텍스트에 언급되면 태그가 지정된 댓글에서 사용자 이름이 강조 표시됩니다.

자세한 내용은 [작업 업데이트](/help/quicksilver/workfront-basics/updating-work-items-and-viewing-updates/update-work.md)를 참조하세요.

## System Activity tab on the new commenting experience supports read-only comments

>[!NOTE]
>
>미리보기 릴리스: 2024년 2월 22일, 빠른 릴리스를 위한 프로덕션: 24.3 릴리스 포함(2024년 3월 14일), 모든 고객을 위한 프로덕션: 24.4 릴리스 포함(2024년 4월 11일)

레거시 주석 달기 환경에서 시스템 활동 레코드에 대한 주석은 이제 새 주석 달기 환경에서 시스템 활동 탭에 읽기 전용으로 채워집니다. 새 주석 달기 환경에서 시스템 활동 레코드에 회신할 수 없습니다.

자세한 내용은 [업데이트 섹션 개요](/help/quicksilver/workfront-basics/updating-work-items-and-viewing-updates/updates-tab-overview.md)를 참조하세요.

## New update stream tab to capture both comments and system activity entries

>[!NOTE]
>
>Preview release: February 22, 2024; Production for fast release: With the 24.3 release (March 14, 2024); Production for all customers: With the 24.4 release (April 11, 2024)

To give you a chronologically cohesive view of both comments and system activity logs, we are introducing a third tab to the Updates area of all objects. The &quot;All&quot; tab captures both user comments and system activity comments in one cohesive stream.

The All tab is a view-only tab. You cannot reply to comments or tag other users in existing comments in the All tab. 특정 댓글에 회신하려면 모두 탭의 댓글 탭에 있는 링크를 사용합니다.

For more information, see [Updates section overview](/help/quicksilver/workfront-basics/updating-work-items-and-viewing-updates/updates-tab-overview.md).
