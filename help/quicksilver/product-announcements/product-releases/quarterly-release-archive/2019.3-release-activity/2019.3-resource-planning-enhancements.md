# 2019.3 리소스 계획 개선 사항

이 페이지에서는 2019.3 릴리스에서 향상된 리소스 계획 기능의 모든 변경 사항에 대해 설명합니다. 프로덕션 환경에서 2019년 8월 19일이 있는 주에 제공됩니다.

2019.3년에 수행된 모든 변경 사항 목록은 [2019.3 릴리스 활동 개요](../../../../product-announcements/product-releases/quarterly-release-archive/2019.3-release-activity/2019.3-release-activity-overview.md).

## 예산책정된 시간에 대한 보고서 작성

이제 Resource Planner 외부에서 예산책정된 시간을 조회하기 위해 보고서를 작성할 수 있습니다.

예산책정된 시간에 대한 보고서를 작성할 때 자원 계획자(예산책정된 시간 필드)의 사용자와 연관된 예산책정된 시간과 프로젝트 또는 작업 역할과 연관된 예산책정된 시간(계획된 예산책정된 시간 필드)을 표시할 수 있습니다.

보고서의 예산책정된 시간 보기에 대한 자세한 내용은 [자원 계획자 예산책정 시간 보고서](../../../../resource-mgmt/resource-planning/report-on-budgeted-hours.md).

## 리소스 계획자에서 실제 시간 열을 숨길 새 설정

이제 리소스 계획자에서 사용자별로 정보를 볼 때 실제 시간 열을 표시하거나 숨길 수 있습니다.

리소스 계획자의 설정에 대한 자세한 내용은 [설정](../../../../resource-mgmt/resource-planning/resource-planner-navigation.md#settings) 섹션 [리소스 플래너 탐색 개요](../../../../resource-mgmt/resource-planning/resource-planner-navigation.md).

## Business Case의 Resource Budgeting 영역에서 자원 풀은 영구적입니다

이제, 비즈니스 케이스의 Resource Budgeting 영역에서 자원 예산을 책정할 때, 프로젝트에 이미 저장한 리소스 풀은 페이지에서 멀어져 다시 돌아올 때 영구적입니다.

업무 사례의 예산 책정 자원에 대한 자세한 내용은 [비즈니스 사례의 예산 자원](../../../../manage-work/projects/define-a-business-case/budget-resources-in-business-case.md).

## 비용 비율에 대한 재무 데이터 액세스는 원가별로 표시되는 자원 계획자의 정보에 영향을 줍니다

이제 Workfront 관리자는 Resource Planner에서 역할, 사용자 또는 프로젝트에 대한 비용 정보를 보거나 편집할 수 있는 액세스 권한이 있는 사용자를 결정할 수 있습니다. 이 기능은 액세스 수준의 재무 데이터 영역에서 사용할 수 있습니다.

이 변경 전에 Resource Planner의 모든 사용자가 항상 모든 비용 정보를 사용할 수 있었습니다.

리소스 계획자의 비용 정보에 대한 자세한 내용은 [리소스 계획자에서 비용 계산](../../../../resource-mgmt/resource-planning/calculate-costs-resource-planner.md).
