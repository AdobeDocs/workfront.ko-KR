# 2018.2 릴리스 활동 개요

>[!NOTE]
>
> 18.2 릴리스는 다음 날짜에 프로덕션 환경에서 사용할 수 있게 되었습니다.
* Workfront 증명 프로덕션 환경 - 7월 17일 화요일
* 클러스터 2 및 4용 Workfront 프로덕션 환경 - 7월 19일 목요일 
* 클러스터 1, 3 및 5용 Workfront 프로덕션 환경 - 7월 20일 금요일

### TLS 지원 종료 중 {#tls-support-is-ending}

최적의 보안을 제공하려면 Workfront에서 TLS 1.1 이상을 사용하도록 TLS 1.0 이전 버전을 사용하는 모든 통합 및 웹 트래픽을 업그레이드해야 합니다.

자세한 내용은 [Adobe Workfront에서 TLS 1.2 필요](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### Flash 애플리케이션 제거 {#flash-application-removal}

Flash 기술을 사용하는 모든 기능은 2018.3 릴리스로 Workfront에서 제거되고 새 솔루션으로 대체됩니다.

이 기능에는 다음이 포함됩니다.

* 기존 Gantt 차트가 새 Gantt 차트로 바뀝니다.

   자세한 내용은 [간트 차트에서 정보 보기](../../../../manage-work/gantt-chart/use-the-gantt-chart/view-info-in-gantt.md).

* Portfolio 최적기가 새 Portfolio 최적기로 대체됩니다

   자세한 내용은 [Portfolio 최적화 프로그램 개요](../../../../manage-work/portfolios/portfolio-optimizer/portfolio-optimizer-overview.md).

* 능력 계획자가 새 자원 계획자로 대체됩니다

   자세한 내용은 [Adobe Workfront의 리소스 계획](../../../../resource-mgmt/resource-planning/resource-planning-overview.md).

### Microsoft Outlook 추가 기능 지원이 종료되고 있습니다. {#microsoft-outlook-add-in-support-is-ending}

Workfront은 7월에 18.2 제품 릴리스 이후에 추가 기능의 데스크탑 버전을 더 이상 지원하지 않습니다.

이제 Workfront for Outlook으로 전환하여 생산성을 높일 차례입니다! 이제 Outlook용 두 가지 고유한 추가 기능을 제공하는 대신 모든 환경에서 사용할 수 있는 하나의 범용 추가 기능을 제공합니다. Outlook 모바일 앱에서도 PC, Mac, 데스크탑, 웹 등을 사용할 수 있습니다.

모든 환경에 새로운 Outlook용 Workfront을 설치하면 팀이 보다 효율적으로 작업을 수행하고, 체계적이며, 작업에 대한 우선 순위를 실시간으로 지정할 수 있습니다.

다음 리소스를 사용하여 Outlook용 Workfront에 대해 자세히 알아보고 시작하십시오.

* [웹 사이트 및 비디오](https://www.workfront.com/integrations/outlook) 
* [Outlook에서 Workfront 사용](../../../../workfront-integrations-and-apps/using-workfront-with-outlook/workfront-for-outlook.md) (도움말 문서)

### 이전 API 버전 제거 {#removing-old-api-versions}

API를 향상시키기 위한 지속적인 노력의 일환으로 최신 Workfront 릴리스와 더 이상 호환되지 않는 이전 API 버전을 제거하고 있습니다.
