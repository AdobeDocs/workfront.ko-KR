# 2017.3 릴리스 활동 개요

이 페이지에서는 2017.3 릴리스에 포함된 기능에 대한 정보를 제공합니다. 

이 페이지에 설명된 기능은 현재 미리 보기 환경에서 사용할 수 있으며, 11월 프로덕션 환경에서 사용할 수 있습니다.  

>[!IMPORTANT]
>
> 미리 보기 환경의 기능은 프로덕션 환경에서 릴리스하기 전에 변경될 수 있습니다.

2017.3 릴리스에는 Workfront 관리자와 다른 사용자를 위한 개선 사항이 모두 포함되어 있습니다.

>[!NOTE]
이 섹션에 설명된 기능을 이제 Workfront Production 계정에서 사용할 수 있습니다. 미리 보기 기능이 아닙니다.

다음 기능이 2017년 8월에 Android 및 iOS 둘 다에 대한 Workfront 모바일 앱의 프로덕션 버전에 추가되었습니다.

* 요청 제출
* 작업표 새 항목
* 사용자 지정 양식 편집
* 증명 승인 요청
* 구독한 항목에 대한 푸시 알림

이러한 변경 사항에 대한 자세한 내용은 [iOS 및 Android용 모바일 앱 업데이트(2017년 8월 초)](../../../announcements/announcement-archive/updated-mobile-app.md).

다음 추가 기능은 11월 8일이 있는 주에 프로덕션 릴리스 전에 사용할 수 있습니다.

* 프로젝트 목록에 액세스
* 프로젝트의 작업 및 하위 작업 목록에 액세스합니다
* 프로젝트의 문제 목록 액세스
* 프로젝트에 새 문제 기록
