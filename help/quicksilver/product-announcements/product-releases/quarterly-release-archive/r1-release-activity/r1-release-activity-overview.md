# R1 릴리스 활동 개요

이 페이지에서는 R1 릴리스에 포함된 기능에 대한 정보를 제공합니다. 

R1 릴리스는 현재 미리보기에서 테스트할 수 있습니다. R1 릴리스는 2017년 4월 26일 및 27일 저녁(MDT)에 단계적으로 프로덕션 환경에 배포됩니다.

R1 릴리스를 통해 사용자, 기술 전문가 및 관리자가 작업을 보다 효율적으로 관리할 수 있습니다. R1을 통해 관리자는 조직의 용어에 맞게 Workfront을 사용자 정의할 수 있습니다. 지식 전문가는 활용 보고서를 활용하여 프로젝트가 계획 및 예산 시간에 대해 추적하는 방법을 더 잘 이해할 수 있습니다. 사용자는 Outlook 365에서 직접 Workfront에 참여할 수 있으며 Workfront 모바일 앱을 사용할 때 더 생산적일 수 있습니다.

## R1에서 기능이 프로덕션으로 릴리즈되지 않음 {#functionality-not-releasing-to-production-in-r1}

이 섹션에 설명된 기능은 R1 릴리스 주기 동안 미리보기 환경에서 사용할 수 있었지만, 이 기능은 R1의 프로덕션에 릴리스되지 않습니다. 6주 동안 미리보기 환경에서 기능이 제거되어 프로덕션에 R1 릴리스가 제공됩니다. 프로덕션에 R1 릴리스 후 미리보기 환경에 다시 도입됩니다.

* [새 리소스 계획 기능(R1의 프로덕션에서 사용할 수 없음)](../../../../product-announcements/product-releases/quarterly-release-archive/r1-release-activity/r1-preview-5.md#new-resource-planning-functionality)(R1의 프로덕션에서 사용할 수 없음; **미리 보기**&#x200B;에서 남음)

* [대시보드 목록에서 업데이트된 모양과 느낌](../../../../product-announcements/product-releases/quarterly-release-archive/r1-release-activity/available-in-preview-in-2016.md#updated-look-and-feel-in-the-dashboard-list)(R1의 프로덕션에서 사용할 수 없음, **조기 액세스**&#x200B;에서 유지)
