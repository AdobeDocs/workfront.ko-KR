# 전체적으로 레코드 유형 만들기

<!--<span class="preview">The information on this page refers to functionality not yet generally available. It is available only in the Preview environment for all customers. 레코드 유형을 전역으로 지정하고 Adobe Workfront Planning의 여러 작업 영역에서 사용할 수 있습니다. 먼저 작업 영역에서 글로벌 레코드 유형을 정의해야 다른 작업 영역에서 가져올 수 있습니다.

Workfront Planning에서 레코드 유형을 만드는 방법에 대한 자세한 내용은 [레코드 유형 만들기](/help/quicksilver/planning/architecture/create-record-types.md)를 참조하십시오.

## 글로벌 레코드 유형 만들기

1. [레코드 형식 만들기](/help/quicksilver/planning/architecture/create-record-types.md) 문서에 설명된 대로 레코드 형식을 수동으로 만들기 시작합니다.


## 기존 글로벌 레코드 유형에서 레코드 유형 만들기

1. 문서 [레코드 종류 만들기](/help/quicksilver/planning/architecture/create-record-types.md)에 설명된 대로 레코드 종류 만들기를 시작한 다음 **글로벌 레코드 종류 사용**&#x200B;을 클릭합니다. <!--check this - the option might have been renamed in the UI-->
