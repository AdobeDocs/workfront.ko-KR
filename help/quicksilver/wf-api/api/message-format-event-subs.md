# 이벤트 가입에 대한 아웃바운드 메시지 포맷

Adobe Workfront 2017.3이 릴리스되면 이벤트 가입에 대한 아웃바운드 메시지 형식이 변경되어 성능을 개선하고 통합에 대한 이벤트 구독을 더 잘 사용할 수 있습니다.

## 아웃바운드 메시지 형식에 대한 업데이트

아웃바운드 메시지 형식은 다음과 같이 변경됩니다.

* 아웃바운드 메시지에는 Workfront 리소스에 대한 oldState 및 newState가 포함됩니다.

   이러한 값은 Workfront의 이벤트 결과로 개체에 수행된 변경 사항을 표시합니다.

* eventTime 메타데이터 필드가 모든 아웃바운드 메시지에 추가됩니다.

   이 필드는 이벤트가 발생한 시간을 나노초 및 EpochSeconds로 나타냅니다. 통합에서 처리한 이벤트를 정렬할 때 eventTime을 사용합니다.

* NOTE 리소스의 owner:companyID 참조 필드가 제거됩니다.
* currentVersion 참조된 개체DOCX(문서) 리소스에서 제거됩니다.

현재 Workfront 이벤트 구독을 사용하는 경우 이러한 변경 사항을 처리하려면 2017.3 릴리스 전에 Workfront 통합을 업데이트해야 합니다.

이벤트 구독에 대한 자세한 내용은 [이벤트 구독 API](../../wf-api/general/event-subs-api.md).
