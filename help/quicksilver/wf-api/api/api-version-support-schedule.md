# API 버전 관리 및 지원 예약



Adobe Workfront은 새로운 API 버전을 정기적으로 출시하며, 일반적으로 일년에 두 번씩 릴리스됩니다. API 버전은 릴리스 후 3년 동안 지원됩니다. 3년 이전 API 버전은 더 이상 지원되지 않습니다.
