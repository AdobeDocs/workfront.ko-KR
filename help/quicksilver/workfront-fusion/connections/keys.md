# Adobe Workfront Fusion의 키

[!DNL Adobe Workfront Fusion]의 키 체인은 공개 및 개인 키를 관리하는 데 도움이 됩니다. 예를 들어 [!UICONTROL Encryptor] 앱에서 PGP 메시지를 암호화하거나 해독하는 데 키를 사용합니다.

## [!DNL Workfront Fusion]에서 키 사용

모듈에 공개 또는 개인 키가 필요한 경우 키 체인에 추가할 수 있습니다. 자세한 내용은 [모듈 설정 구성 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/modules/configure-a-modules-settings.md)을 참조하세요.


공개 키와 개인 키에 대한 자세한 내용은 [기본 용어 [!DNL Adobe Workfront Fusion]](../../workfront-fusion/get-started/basic-terms.md)를 참조하십시오.
