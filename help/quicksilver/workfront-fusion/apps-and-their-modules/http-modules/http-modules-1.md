# HTTP 모듈

>[!NOTE]
>
>[!DNL Adobe Workfront Fusion] 를 사용하려면 [!DNL Adobe Workfront Fusion] 라이센스 [!DNL Adobe Workfront license].

HTTP 앱은 HTTP(Hypertext Transfer Protocol) 프로토콜을 기반으로 통신을 위한 다양한 모듈을 제공합니다. HTTP는 World Wide Web을 위한 데이터 통신의 기반입니다. 모듈을 사용하여 웹 페이지 및 파일을 다운로드하고, 웹 후크 및 API 엔드포인트 등을 호출할 수 있습니다.

모듈의 올바른 선택은 액세스하려는 리소스가 사용하는 인증/인증 메커니즘에 따라 다릅니다.
